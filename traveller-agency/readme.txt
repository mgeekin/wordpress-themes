== Description ==

Traveller Agency Responsive Travel Management Software for Travel and Tourism Agency Websites. Multipurpose theme design for travel, honeymoon booking, family vacations, vacation planning, adventure, blog, blog theme blogger, personal, private blog, responsive, and tourism. You can have a blog, gallery, and fully responsive design. There are unlimited layout options, a press forum, tons of short codes, and a Mega Menu. This package type runs on Woo commerce and allows you to display all package packages for your customers Customizing, managing, and searching for hotels, places, and a gallery with powerful custom post types for Hotels, Places, and galleries is easy. We plan to add more modules for booking, listing, and promoting. This theme is packed with features and can be used to build almost any kind of Travel business. You can sign in using the following credentials. Take a look inside to see how users will view your website. Explore all the functionalities and options, including adding new tours, editing and managing your account, and many more. It is a beautifully designed theme that catches the user's attention at first glance. This theme is intuitive thanks to fresh colors and appropriate typography, making it even more effective. This theme have a modern and contemporary outlook that encourages visitors to surf and explore. The inner pages, which are well-designed and spacious, provide complete information about the business and product.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Traveller Agency in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
